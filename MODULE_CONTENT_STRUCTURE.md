## ğŸ“Š **ASSESSMENT SYSTEM**

### **1. Pre-Test (Placement Test)**
**Purpose**: Determine starting level and module priority
- **Duration**: 20-30 minutes
- **Questions**: 40 questions (10 per category)
- **Categories**:
  1. Oral Language (10 questions)
  2. Word Knowledge (10 questions)
  3. Reading Comprehension (10 questions)
  4. Language Structure (10 questions)

**Output**:
- Performance score per category (0-100%)
- Placement level (Low/Mid/High Grade 3)
- Module priority order (weakest â†’ strongest)
- Modules unlocked based on placement

### **2. Lesson Quizzes**
**Purpose**: Check understanding after each lesson
- **Frequency**: After every lesson
- **Questions**: 5-10 questions
- **Passing Score**: 70%
- **Retakes**: Unlimited (review lesson if failed)

### **3. Module Assessment**
**Purpose**: Verify mastery of entire module
- **When**: After completing all lessons in a module
- **Questions**: 20-30 comprehensive questions
- **Passing Score**: 80%
- **Retakes**: Up to 3 attempts
- **Requirement**: Must pass to unlock next module

### **4. Post-Test (Final Assessment)**
**Purpose**: Measure overall growth and learning
- **When**: After completing all 5 modules
- **Format**: Same as pre-test (40 questions, same categories)
- **Comparison**: Direct comparison with pre-test results

**Output**:
- Growth percentage per category
- Overall improvement score
- Certificate of completion
- Detailed progress report
