# Sound Files Directory

## Required Sound Files (MP3 format)

Add these sound files to this directory:

1. **button_click.mp3** - Used when clicking buttons in WelcomeActivity
2. **correct_answer.mp3** - Play when answer is correct
3. **wrong_answer.mp3** - Play when answer is incorrect
4. **level_up.mp3** - Play when user levels up
5. **achievement.mp3** - Play when earning badges/achievements
6. **game_start.mp3** - Play when starting a game
7. **game_complete.mp3** - Play when completing a game/module

## Notes:
- All files should be in MP3 or OGG format
- Keep file sizes under 500KB for better performance
- Use 44.1kHz sample rate
- Android Studio may show "not associated with file type" warning - this is normal, ignore it
